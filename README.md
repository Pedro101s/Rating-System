# Spring Boot Rating System
A platform to where movie watchers, movie goers and critics go and rate different movies.

</div>

# Features
- CRUD operations for managing Reviews (Create, Read, Update, Delete).
- Rating system allowing users to rate posts.
- User-friendly interface for navigating posts and rating them.

# Introduction 
A CRUD (Create, Read, Update, Delete) application using Spring Boot, 
with an added rating system feature where users can perform basic CRUD operations on posts and rate them.

# Tech Stack
- Frontend: Html, Css, JavaScript, Thymeleaf
- Backend: Spring Boot, Spring Data JPA
- Database: POSTGRESQL
- Other Tools: Github (Version control), Docker, Postman

# Data Interpretation
User do not have to be logged in to see the main page and the post pages
